Training

If you are new to disease modeling, start here. These tutorials guide you through modeling a variety of disease scenarios using the EMOD Quick Start.

General Disease

The EMOD generic simulation type models person-to-person disease transmission and intervention. It can be configured to model a variety of diseases, such as typhoid, measles, influenza and more.

HIV

The EMOD HIV documentation includes information about HIV biology, HIV prevention and treatment, and HIV tutorials that illustrate how to use the configuration and campaign parameters.

Malaria

The EMOD malaria documentation includes information about malaria biology, the mosquito lifecycle, malaria interventions, and parameters available to use when modeling malaria.

TB

The EMOD tuberculosis documentation includes information about TB biology, TB prevention and treatment, multidrug-resistant TB, and TB tutorials that illustrate how to use the configuration and campaign parameters.
