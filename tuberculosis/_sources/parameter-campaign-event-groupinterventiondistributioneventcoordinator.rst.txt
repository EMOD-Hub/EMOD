=============================================
GroupInterventionDistributionEventCoordinator
=============================================

The **GroupInterventionDistributionEventCoordinator** coordinator class is similar to
:doc:`parameter-campaign-event-standardinterventiondistributioneventcoordinator`, but allows individuals to be targeted by
disease state (the **Target_Demographic** parameter must be set to ExplicitDiseaseState). Disease
state cannot be combined with age or gender targeting. See the following JSON example and table,
which shows all available parameters for this event coordinator.
