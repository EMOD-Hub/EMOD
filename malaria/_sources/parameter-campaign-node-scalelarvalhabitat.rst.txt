==================
ScaleLarvalHabitat
==================

The **ScaleLarvalHabitat** intervention class enables species-specific habitat modification within shared habitat
types. This intervention has a similar function to the demographic parameter
**ScaleLarvalMultiplier**, but enables habitat availability to be modified at any time or at any
location during the simulation, as specified in the campaign event.
